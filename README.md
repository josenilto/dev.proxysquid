# dev.proxysquid
O Squid é um proxy da web HTTP de cache e encaminhamento. Ele tem uma ampla variedade de utilizações, incluindo acelerar um servidor web ao armazenar em cache solicitações repetidas, caching web, DNS e outras pesquisas de rede de computador para um grupo de pessoas que compartilham recursos de rede e auxiliar na segurança ao filtrar o tráfego.
